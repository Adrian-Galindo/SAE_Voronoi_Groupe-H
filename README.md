#  SAE S6 — Diagrammes de Voronoï | Groupe H

> **BUT Informatique · Université Sorbonne Paris Nord (USPN) · 2026**

---

## 📌 Objectif de la SAE

Nous devions lors de cette SAE concevoir des applications pour générer et visualiser un diagramme de Voronoï à partir d’un ensemble de points du plan de coordonnées .


## Phase 1
Nous avons proposé une interface conviviale à la main qui permet à partir de fichiers qui contiennent une liste de points sous forme de paire de nombres (les coordonnées) au rythme de une par ligne :

```
2,4
5.3,4.5
18,29
12.5,23.7
```

L'application permettra de visualiser le diagramme obtenu et éventuellement exporter le résultat sous forme de fichiers SVG ou image.

Cette partie à été faite sans aucune aide d'aucune sorte de la part d'intelligence artificielles.

## Phase 2
Après avoir terminé la phase 1, nous avons développer avec diverses IA génératrices de code la réalisation d'une application avec le même cahier des charges. L'application fournie par l'IA inclue donc aussi notamment une série de tests.

## Phase 3
Cela regroupe nos 6 rapport d'analyse d'un thèmes sur L'IA dans le développement informatique
